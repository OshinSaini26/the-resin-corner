# The Resin Corner

A full-stack resin art ordering platform developed for small resin businesses.

## Features

- Customer Registration and Login
- Seller Login and Dashboard
- Ready-Made Product Ordering
- Custom Resin Product Ordering
- Custom Pricing Workflow
- Advance and Remaining Payment Tracking
- Order Status Tracking
- Product Image Gallery with Zoom
- MySQL Database Integration

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL
- Version Control: Git & GitHub
